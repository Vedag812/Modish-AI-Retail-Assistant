## How to Use in Presentations

1. **Copy the ASCII diagram** you want
2. **Paste into a text box** in PowerPoint/Google Slides
3. **Use a monospace font** (Consolas, Courier New, or Fira Code)
4. **Adjust font size** (14-18pt works well)
5. **Center align** the text box

### Recommended Fonts:
- **Consolas** (Windows)
- **Monaco** (Mac)
- **Fira Code** (Cross-platform)
- **JetBrains Mono** (Modern look)
